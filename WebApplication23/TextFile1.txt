Задание 


Создайте проект типа ASP.NET MVC Web Application. 
Создайте модель которая будет хранить информацию о пользователе. 
Используя шаблонизированные методы создайте форму для добавления пользователей. 
На сервере, полученные со стороны клиента значения, выводите в output окно 
с помощью Debug.Write.
