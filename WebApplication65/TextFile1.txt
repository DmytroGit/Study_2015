Задание 2 
Создайте приложение, которое будет использовать модель из проекта #2 
текущего урока. 

Создайте форму, 
которая будет выводить 

всех покупателей, 

которые 
содержатся в модели, в виде ссылок на странице. 

Реализуйте форму таки образом, чтобы при нажатии 
по ссылке происходил запрос на сервер, благодаря 
которому, будет отображаться детальная информация 
о покупках совершенных выбранным пользователем. 
Информация должна запрашиваться асинхронно и 
запрос не должен приводить к перезагрузке страницы.
