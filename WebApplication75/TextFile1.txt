Задание 
Создайте проект по шаблону Basic MVC4 Web Application. 
Сделайте метод действия, который будет выдавать в ответ 
пользователю представление с сообщением 

«данные не предоставлены» 

если запрос сделанный к контроллеру не содержит параметра 

id
 
переданного через сегменты. Если параметр есть, метод действия 
должен возвращать строку 

(plain/text) со значением, которое находится в сегменте id.
